# 推奨コマンド

- `pnpm install` : 依存ライブラリのインストール（推奨パッケージマネージャ）。
- `pnpm dev` : Next.js 開発サーバーを起動。
- `pnpm build` : 本番ビルドを生成。
- `pnpm start` : ビルド済みアプリを本番モードで起動。
- `pnpm lint` : ESLint (`next/core-web-vitals` ベース) を実行。
- `git status` / `git diff` : 作業ツリーと変更内容の確認。
- macOS 共通: `ls`, `pwd`, `cd`, `grep`, `find` などの基本ユーティリティ。
